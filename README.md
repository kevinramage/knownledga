# Knownledga

Knownledga is  a tool to read and write markdown files.

## Installation

Install dependencies:

```
npm update
npm update -D
npm install -g pnpm
```

Download rust

## Run and build

Run client
```
pnpm tauri dev
```

Build a release
```
pnpm tauri build
```
